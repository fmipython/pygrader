Functionality
=============

The grader shall provide the following functionality:

- Check if the project has a requirements.txt file
- Score the code for PEP-8 compliance
- Score the code based on type-hints usage
- Score the code based on tests code coverage
- Support configuration files for specifying the checks to be executed
- Support configuration files for specifying the score for each check
- All output is collected in a log file
- Accept a student's project path as an argument
- Accept a student's ID as an argument
- Accept a configuration file as an argument
- Run certain checks inside a virtual environment
- Detect source and test directories
